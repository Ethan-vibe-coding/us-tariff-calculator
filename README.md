# 中国输美产品关税叠加计算查询系统

输入美国HTS税号（8位或10位），自动计算 **最惠国税率（MFN）＋ 301 ＋ 232 ＋ Section 122** 各项加征及总叠加值，逐项列示税率、加征值与法律依据，并自动适用"122与232不叠加"等规则。

**在线使用**：打开 `index.html` 即可（纯静态页面，无需服务器）。

## 功能

- **关税叠加计算器**：单码实时计算，含232"15%封顶"、从量税提示等规则
- **批量计算**：多税号一次计算，可导出CSV
- **数据库浏览**：MFN全表（11,414）、301清单（10,008＋战略产业392）、232措施（1,061）、122例外（1,097）、301排除（178）在线检索
- **数据更新中心**：一键检查联邦公报最新官方文件；支持导入hts.usitc.gov官方文件本地比对（第99章税目/注释指纹/MFN全表），MFN差异可直接应用并导出更新后的数据文件
- **数据下载**：SQLite数据库（us_china_tariffs.db，8张表）

## 数据基准

2026-07-18（HTS 2026HTSRev11）。数据源：hts.usitc.gov（HTS第1—97章及第99章第三分章注释2/16/20/31/33/37/38/39）、USTR联邦公报公告、总统公告10895/10896/10908/10962/11012、CBP CSMS指引。
